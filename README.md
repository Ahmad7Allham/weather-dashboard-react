# Weather Dashboard 🌤️
Modern weather dashboard built with React and Tailwind CSS.
## Live Demo  https://app-weather-dashboard.netlify.app/

## Features

- Search weather by city
- Real-time weather data
- Dark Mode
- Responsive Design
- Smooth animations
- LocalStorage support

## Technologies

- React
- Tailwind CSS
- Framer Motion
- WeatherAPI

## Installation

```bash
npm install
npm start
```
